## 项目概览

这份《微电路设计小组报告》围绕室内扫地机器人平台展开，内容覆盖硬件电路设计、嵌入式软件控制、上位机导航协同、功能测试与项目管理。
系统整体采用 “STM32 下位机实时控制 + 树莓派桥接 + PC 上位机规划” 的分布式架构，把电机驱动、传感器采集、导航规划、语音与网页交互以及可视化监控串联成一套完整工程方案。

## 主要内容

- 以 `STM32F407ZGT6` 为核心完成电源、驱动、传感与外设接口的集成电路设计，并形成原理图、PCB 与 3D 设计输出。
- 在下位机侧构建基于 `HAL + BSP + Module + Application + Task` 的分层软件架构，结合 `FreeRTOS` 实现多任务调度。
- 在上位机和树莓派侧使用 `ROS 2`、`DDS`、`Nav2` 与 `RViz` 搭建导航系统，实现建图、定位、路径规划、动态避障与回充流程。

## 阅读摘要

报告的核心价值在于把板级硬件设计、嵌入式实时控制和上位机导航协同完整串联起来，体现出较清晰的系统工程视角。
除了硬件集成与任务调度外，条目也保留了多模态交互、可视化监控和后续调优的扩展空间。

## 进展 / 结果

- 已完成从硬件电路、嵌入式控制到 ROS 2 导航协同的完整方案梳理。
- 形成了带有分工说明、测试记录与总结反思的课程小组报告。
- 后续可以继续补充模块兼容性评估、参数整定效率和工程联调节奏相关内容。
