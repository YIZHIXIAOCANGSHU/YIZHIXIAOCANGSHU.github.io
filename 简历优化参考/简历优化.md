简历自检规范
各位同学好，近期暂时停止简历接单服务，大家可以先按照以下统一规范完成简历的自检与优化，确保内容结构完整、重点突出、匹配岗位需求。
一、简历固定撰写顺序
基本信息 → 教育背景 → 实习经验→ 项目经验 → 专业技能 → 荣誉奖项
二、各模块详细撰写规范
1. 基本信息
- 必填核心项：姓名、所在城市（无需详细住址）、手机号、常用邮箱号
- 必填项：求职意向，包含意向岗位、意向城市，可补充期望到岗时间、薪资范围
- 补充项：如有作品集、GitHub 主页、个人技术博客，附上对应链接
2. 教育背景
- 入学时间 - 毕业时间 | 学校名称 | 学院 | 专业 | 学历
- 补充：GPA 较高（建议专业前 30%）可标注 GPA 及专业排名；综测排名较高（建议专业前 20%）可标注综测排名及对应比例；应届生可补充与岗位匹配的核心专业课程。
3. 实习经历
- 开头必须标注「实习开始时间 - 结束时间 | 公司名称 | 担任的身份 / 角色（以合同为准）」
- 从新到旧处理，有几个写几个，挤压到项目就删项目。
4. 项目经历
- 基础格式强制要求：每个项目开头必须标注「项目开始时间 - 结束时间 | 项目名称 | 项目中担任的身份 / 角色」
- 数量要求：至少撰写 3 个核心项目，优先排序与求职岗位匹配度最高的项目
  - ①项目中使用的核心技术 / 工具 / 方案；
  - ②你针对性解决了哪些具体、可落地的问题 / 痛点；
  - ③问题解决后带来了哪些可量化的提升 / 成果 / 价值。
- 项目核心内容建议分点撰写，单条内容聚焦一个技术动作 + 问题解决 + 量化成果，逻辑更清晰。

- 核心设计能力（传动系统设计、公差与配合设计、结构轻量化设计、钣金设计、注塑模具设计等）、力学分析能力（静力学分析、强度 / 刚度校核、疲劳寿命分析等）

1. 荣誉奖项
- 单条奖项固定为「获奖时间 奖项名称 奖项级别」，单条一行
- 按奖项级别从高到低、获奖时间从近到远排序
- 奖项级别统一标注为国家级 / 省级 / 市级 / 校级 / 院级；如获奖有排名、获奖比例，可补充标注（例：2024.09 全国大学生机械创新设计大赛 国家级一等奖 全国前 5%）
- 优先放与求职岗位相关的技术类、学科竞赛类奖项，弱化与岗位无关的奖项。
